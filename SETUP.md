# সামসুল'স গ্রোসরি - সেটআপ গাইড
## Samsul's Grocery - Setup Guide

এই প্রকল্পটি একটি পূর্ণ-কার্যক্ষম অনলাইন গ্রোসারি স্টোর, যা Vite + React + Supabase দিয়ে তৈরি।

### প্রাক-প্রয়োজনীয়তা (Prerequisites)
- Node.js 16+ এবং npm/pnpm
- একটি Supabase অ্যাকাউন্ট (https://supabase.com)

---

## ১. ইনস্টলেশন (Installation)

```bash
# ক্লোন করুন অথবা ডাউনলোড করুন
git clone <repository-url>
cd samsul-dokan

# ডিপেন্ডেন্সি ইনস্টল করুন
pnpm install
# অথবা npm install
```

---

## ২. Supabase সেটআপ (Supabase Setup)

### ধাপ ১: প্রজেক্ট তৈরি করুন
1. https://supabase.com এ যান এবং নতুন প্রজেক্ট তৈরি করুন
2. আপনার প্রজেক্টের **Project URL** এবং **Anon Key** নোট করুন
   - Settings → API → Project URL, anon key এ পাবেন

### ধাপ ২: Supabase.ts আপডেট করুন
`supabase.ts` ফাইলটি খুলুন এবং আপনার Supabase URL এবং Key দিয়ে প্রতিস্থাপন করুন:

```typescript
const SUPABASE_URL = 'your_project_url_here';
const SUPABASE_ANON_KEY = 'your_anon_key_here';
```

### ধাপ ৩: ডাটাবেস তৈরি করুন
1. Supabase ড্যাশবোর্ড খুলুন
2. **SQL Editor** এ যান
3. `scripts/setup-database.sql` থেকে সম্পূর্ণ SQL কপি করুন
4. SQL Editor এ পেস্ট করুন এবং চালান (Execute)

এটি স্বয়ংক্রিয়ভাবে তৈরি করবে:
- `products` টেবিল
- `slides` টেবিল
- `orders` টেবিল
- প্রয়োজনীয় ইনডেক্স এবং রিয়েল-টাইম সাবস্ক্রিপশন

### ধাপ ৪: স্টোরেজ বাকেট তৈরি করুন (ঐচ্ছিক - পণ্যের ছবির জন্য)
1. Supabase ড্যাশবোর্ড → **Storage**
2. নতুন বাকেট তৈরি করুন, নাম দিন: `images`
3. **Public** সেট করুন (পাবলিক অ্যাক্সেস)
4. নীতি সেট করুন যাতে সবাই পড়তে পারে

## ৪. অ্যাডমিন প্যানেল অ্যাক্সেস (Admin Panel Access)

অ্যাডমিন প্যানেল অ্যাক্সেস করতে:

1. হেডার → **অ্যাডমিন** ক্লিক করুন (বা `/admin` যান)
2. **পাসওয়ার্ড লিখুন**: `admin123`
3. এন্টার দিন বা লগইন বাটন ক্লিক করুন

### অ্যাডমিন প্যানেল ফিচার:
- **ড্যাশবোর্ড**: বিক্রয় ও অর্ডার পরিসংখ্যান
- **পণ্য তালিকা**: পণ্য যোগ, সম্পাদনা, মুছুন
- **স্লাইডার**: হোমপেজ স্লাইড পরিচালনা করুন
- **অর্ডারসমূহ**: গ্রাহক অর্ডার ট্র্যাক করুন, অবস্থা আপডেট করুন
- **সেটিংস**: ডাটাবেস সেটআপ ও পুনরুদ্ধার

## ৭. Troubleshooting

### সমস্যা: "ডাটাবেস টেবিল পাওয়া যায়নি"
**সমাধান**: Supabase SQL Editor এ `setup-database.sql` চালান

### সমস্যা: পণ্যের ছবি আপলোড হচ্ছে না
**সমাধান**: 
- Supabase ড্যাশবোর্ড → Storage → `images` বাকেট তৈরি করুন
- Public অ্যাক্সেস নিশ্চিত করুন

### সমস্যা: অ্যাডমিন প্যানেল পাসওয়ার্ড ভুল
**সমাধান**: AdminPanel.tsx এ পাসওয়ার্ড পরিবর্তন করুন (লাইন ~208):
```typescript
if (password === 'আপনার_নতুন_পাসওয়ার্ড') setIsAuthenticated(true);
```

### সমস্যা: Real-time আপডেট কাজ করছে না
**সমাধান**: Supabase ড্যাশবোর্ড → Replication এ নিশ্চিত করুন সব টেবিল সক্ষম আছে
